# [컨셉] <a name='1'></a>

## 메인컨셉 : 타임어택 

- 스테이지형식 타임어택류 RPG 액션 게임 

### 서브컨셉 1  :  

- 타이머 / 시간이 흐르며 최단시간을 랭킹에 등록하여 사람들과 경쟁할 수 있게끔 만듦.


### 서브컨셉 2 : 

- 라운드화 / 라운드 마다 적들의 체력및 이동속도가 빨라지게 하고, 특정 라운드 마다 보스들을 배치시켜 잡지 못하면 스테이지를 넘어가지 못하는 컨셉을 가지게 만듬. 


### 서브컨셉3 : 

- 스테이터스 / 스테이터스를 자유롭게 찍으며 자신만의 클리어 방법을 찾는 재미를 얻게함. 


### 서브컨셉4 : 2.5D  

- Lowpoly / 2.5D LowPoly RPG 액션게임 




# [구성 요소] <a name='2'></a>
 - 스테이지형식 타임어택류 RPG 액션 게임 

## 1. 메커니즘 



### [도전 과제] 

- 스테이지 마다 나오는 적들을 사살하여 최종 스테이지까지 클리어한다.

- 적들을 잡아 나오는 경험치로 레벨업을 하며, 캐릭터 성장을 하여 더 쉽게 클리어 하게 한다. 

- 자신이 세운 시간을 보며 랭킹순위에 등록된 시간보다 더욱더 최단시간을 뽑는 도전정신을 이끈다.



### [재미 요소]

- 랭킹을 세움으로서 다른사람들과의 대결구도를 만들 수 있다. 

- 스텟을 자유롭게 찍음으로서 자신만의 스타일을 만들 수 있다. 





## 2. 미적요소



### [디자인 및 컬러]

LOW 퀄리티의 캐릭터 에셋 및 에너미 오브젝트를 씀으로서 아기자기한 느낌으로 모든 연령층이 편하게 

접근 할 수 있도록 생각했습니다. 



### [음향]

무료 음원 사이트를 참고하여 적을 공격, 이동 등에 대해서 타격감이 있는 플레이를 할 수 있습니다.



## 3. 기술

Unity Engine 을 통해 게임을 제작 

저사양으로 플레이 가능하도록 개발할 예정이다.

- ### [게임의 규칙]<a name='3f'></a>
#### 1) 핵심 규칙

 - 플레이어의 체력이 0이 될 시, 게임 실패 처리를 한다. 
 - 몬스터를 잡을 시 일정 경험치를 얻는다. 
 - 총 3스테이지 까지 존재하며, 몬스터의 종류는 1스테이지 3종류, 2스테이지 4종류, 3스테이지 4종류 존재한다. 
 - 최종 보스 몬스터를 못 잡을 시 게임이 끝난다. (실패 처리) / 최종 보스 몬스터를 잡을 시 게임이 클리어 한다.
 - 게임 클리어시 랭킹순위에 닉네임과 타이머시간을 1점으로 처리하여 스코어가 계산되어 등수처리를 한다. 


 
#### 2) 보조 규칙

스테이지 3 까지 존재. 

- ### [게임에서 사용될 공식]<a name='5g'></a>
- 초기 스테이터스 기준 레벨업 시 스텟포인트 +3 제공 

- 스테이지 마다 몬스터존재 / 몬스터 처치 시 일정 경험치 획득
- 스테이지 보스 처치 시 다음 스테이지 이동 가능한 포탈 생성 

- 직업 스텟, 공격력/ 체력/ 이동속도 업그레이드 가능 



# [요구사항(1년차)]<a name='4'></a>

- ### [요구사항] <a name='6a'></a>
 Devil Defense 게임의 요구사항
- 시작화면, 시작화면-설정, 게임화면-설정, 게임화면 총 4개의 화면이 있다.
- 시작화면에는 시작하기, 게임설정, 게임종료 버튼이 있다.  
- 게임설정 클릭 시 시작화면-설정 으로 넘어간다.
- 게임종료 클릭 시 게임이 종료된다.
- 시작화면-설정 에는 게임 음악 볼륨을 on, off 하는 버튼, 볼륨 조절 할 수 있는 바, 
게임플레이 설명창 버튼, 뒤로가기 버튼이 있다.
- 게임플레이 설명창에서는 캐릭터 간 이동버튼 설명서와 NPC 이용 방법 설명서 및 뒤로가기 버튼이 있다.
- 시작화면에서 시작하기 클릭 시 게임화면으로 변경된다.


- 이어하기 클릭시, 게임화면으로 돌아간다.
- 처음부터 클릭시, 게임이 처음부터 시작된다.
- 게임플레이 설명창 클릭시 게임플레이 설명창으로 넘어간다.
- 스테이지는 3스테이지 까지 존재한다. 


- 게임 종료 조건은 체력을 남기고 최종보스를 잡아 클리어 하거나, 체력 0이 되어 클리어 실패를 하면 종료되고 팝업이 뜬다. 

- 게임 클리어시 게임 클리어 팝업이 뜨고 랭킹이 등록되며, 시작화면으로 돌아간다.
- 게임 클리어 실패 시 게임 클리어 실패 팝업이 뜨고 시작화면으로 돌아간다.
